Youtube link for the video to help create a python envirnoment: https://www.youtube.com/watch?v=KxvKCSwlUv8 

Documentation for creating new virtual environments: https://docs.python.org/3/library/venv.html 

Creating a virtual envirnoment: 'python -m venv  <name-of-virtual-envirnoment> (optional)

How to activate the "venvEventure" envirnoment: source venvEventure/bin/activate

Your imports that you write in a "requirements.txt" file are imported into your virtual environment python rather
than your global one.

It's always good to keep track of your dependencies by adding them to the "requirements.txt" 

When you're testing things or messing around you can add THOSE dependencies to "requirements-dev.txt"
